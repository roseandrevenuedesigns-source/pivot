# Pivot
A Containment Method workspace built on the Financial Containment Operating System (FCOS)

Pivot is a stabilization-first financial workflow designed for households in transition, recovery periods, or early financial structure-building phases.

Most financial tools begin with optimization. Pivot begins with containment.

Instead of budgeting dashboards or payoff calculators, Pivot provides browser-based monthly containment tools that help users:

- protect housing and utilities
- sequence obligations by paycheck window
- maintain minimum continuity across accounts
- track stabilization progress month-to-month
- visualize bill timing using a calendar-based execution view

Pivot is built on the Financial Containment Operating System (FCOS), a structured containment-first workflow framework designed to support stability before optimization.

## Workflow structure

Pivot currently includes five working containment layers:

Layer 1 — Household Stability Checklist  
Layer 2 — Monthly Bill Map  
Layer 3 — Monthly Credit Snapshot  
Layer 4 — Quarterly Review  
Layer 5 — Annual Review

Together these create a practical containment sequence:

stability → obligations → credit continuity → quarterly reflection → annual direction

This system is designed to support structured adjustment across changing life stages, income shifts, and real-world obligations without requiring traditional budgeting systems.
